macOS 设置

Homebrew 安装

/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

安装软件 brew install 软件名

给MacOS终端CLI添加颜色 Terminal

  • vim ~/.bash_profile
  • 更新刚配置的环境变量 source .bash_profile

ll = ls -al 设置别名

  • vim ~/.bash_profile alias ll='ls -al'

Mac tree 命令，中文文件名乱码

  • tree -N
  • tree -L 2 2层级
